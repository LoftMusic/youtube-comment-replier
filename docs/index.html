How It Works

Simple setup, powerful automation.

1

Authorize Channel

Run the OAuth flow once per channel. Tokens are stored locally — you control access.

2

Add Videos

List the videos you want to monitor in a simple text file. One per line.

3

Configure Reply

Set your reply template — greeting text and link to promote.

4

Run

The tool fetches comments, detects sentiment, and replies to positive ones automatically.
